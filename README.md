🎧 Simulator Spotify

Este projeto é uma simulação da interface do Spotify, desenvolvida com o objetivo de praticar e aprimorar habilidades em desenvolvimento front-end utilizando HTML, CSS e JavaScript.

A aplicação recria elementos visuais e interações do famoso player de música, proporcionando uma experiência próxima à original.
